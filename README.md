# 원형 생활계획표 — PWA 한 벌 (설치/배포 안내)

이 폴더는 "홈 화면에 깔리고 인터넷 없이도 작동하는 앱(PWA)" 한 벌입니다.

## 들어있는 파일
- `index.html` — 앱 본체 (지금까지 만든 원형 시간표 프로그램)
- `manifest.json` — 앱 이름·아이콘·색 정보
- `sw.js` — 오프라인 작동(서비스 워커)
- `icons/` — 앱 아이콘 이미지들

> 중요: PWA 기능(홈 화면 설치·오프라인)은 **웹 주소에 올렸을 때만** 작동합니다.
> 컴퓨터에서 `index.html`을 더블클릭으로 열면 시간표는 동작하지만 "앱 설치"는 안 됩니다.
> 아래처럼 GitHub Pages 같은 곳에 올리면 그때부터 앱처럼 깔 수 있습니다.

---

## GitHub Pages에 올리는 방법 (무료)

1. github.com 에 로그인합니다. (이미 'artist4rhythm-hub' 같은 계정을 쓰고 계셨다면 그걸 사용)
2. 새 저장소(Repository)를 만듭니다. 이름 예: `circle-timetable`
   - "Public"으로 둡니다.
3. 이 폴더의 **모든 파일과 icons 폴더**를 그 저장소에 업로드합니다.
   - 웹사이트에서 "Add file → Upload files"로 끌어다 놓으면 됩니다.
   - `index.html`, `manifest.json`, `sw.js`, 그리고 `icons` 폴더 전체를 올리세요.
4. 저장소의 **Settings → Pages** 로 갑니다.
5. "Build and deployment"의 Source를 **Deploy from a branch**,
   Branch를 **main / (root)** 로 선택하고 Save.
6. 1~2분 뒤 `https://[내아이디].github.io/circle-timetable/` 주소가 생깁니다.
   - 이 주소를 카톡/블로그로 공유하면 누구나 바로 쓸 수 있어요.
   - 휴대폰에서 그 주소로 들어가 "홈 화면에 추가"를 누르면 앱 아이콘이 깔립니다.

---

## 휴대폰에서 앱처럼 설치하기
- **안드로이드(크롬):** 주소로 접속하면 화면에 "📲 앱 설치" 버튼이 뜹니다. 누르면 끝.
  (또는 크롬 메뉴 → "앱 설치 / 홈 화면에 추가")
- **아이폰(사파리):** 공유 버튼(□↑) → "홈 화면에 추가".

설치하면 주소창 없이 전체화면으로 열리고, 인터넷이 끊겨도 시간표를 만들고 인쇄할 수 있습니다.

---

## 앞으로 프로그램을 더 고칠 때
- 고치는 건 항상 `index.html` 파일입니다. (지금까지처럼 기능 추가 가능)
- 수정한 `index.html`을 저장소에 다시 올리면 됩니다.
- **중요:** 내용을 바꿨는데 사용자 화면이 옛날 그대로면, `sw.js` 파일 맨 위
  `const CACHE = "wonhyung-timetable-v1";` 의 숫자를 `v2`, `v3`… 으로 올려서 다시 올리세요.
  그래야 새 버전이 사용자에게 반영됩니다. (오프라인 저장 때문에 생기는 자연스러운 과정입니다)
